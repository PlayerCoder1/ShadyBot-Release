# ShadyBot

**ShadyBot** est un outil gratuit permettant d’analyser les paquets du jeu et d’afficher les informations de manière claire dans une interface simple à utiliser.

Une version **Premium** est disponible pour débloquer les fonctionnalités d’automatisation avancées et soutenir le développement.

## IA personnalisée

ShadyBot permet de créer une IA de combat personnalisée avec des scripts C#.

Il est possible de faire une IA différente selon la map, les monstres, les PA, PM, PV, cooldowns ou distances.

⚠️ L’IA personnalisée est encore très expérimentale et peut nécessiter des ajustements.

## Métiers / Récolte

Le système de métiers permet de détecter et récolter certaines ressources automatiquement selon le trajet configuré.

⚠️ Cette fonctionnalité est encore très expérimentale.
